# Heart Failure Risk Prediction System

A machine learning web application for predicting heart failure risk based on clinical parameters.

## Features

- **Single Patient Prediction**: Input clinical parameters for individual risk assessment
- **Batch Prediction**: Upload CSV files for multiple patient predictions
- **Analytics Dashboard**: Visualize prediction results with interactive charts
- **Clinical Recommendations**: Get personalized recommendations based on risk factors
- **Feature Importance**: Understand which factors contribute most to risk
